4. 运算符

5.条件判断

基本语法

[condition]

-lt小于 -le 小于等于

-eq等于

-gt大于 -ge大于等于

-ne不等于

-r -w -x -f -e -d

判断23是否大于等于22

[ 23 -ge 22 ]

echo $?

&& 表示上一条命令成功后才执行后一条命令。||表示上一条命令失败后才执行下一条命令。

[ -e passwd ] && echo OK || echo notok
OK
[ -e passwd ] && [ ] || echo notok
notok

8.2 sed切割

sed是流编辑器，一次处理一行内容。

sed

-e 直接在指令的列模式上进行增加

a 新增

d 删除

s 查找并替换
