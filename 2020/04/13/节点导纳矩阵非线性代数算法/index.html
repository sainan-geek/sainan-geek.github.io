两种广泛应用的解非线性代数方程的迭代方法：高斯-塞德尔法(Gauss-Seidel)和牛顿-拉夫逊法(Newton- Raphson)。

高斯-塞德尔法

matlab实现高斯塞德尔算法

matlab代码如下：

dx=1;                   % Change in variable is set to a high value 设置一个高值
x=2;                                             % Initial estimate 初始估计值
iter = 0;                                       % Iteration counter 迭代计数
disp('Iter    g         dx        x')         % Heading for results 解的题头
while abs(dx) >= 0.001 & iter < 100          % Test for convergence 测试收敛
iter = iter + 1;                                % No. of iterations 迭代次数
g = -1/9*x^3+6/9*x^2+4/9 ;
dx = g-x;                                      % Change in variable 修正量
x = x + dx;                              % Successive approximation 连续迭代
fprintf('%g', iter), disp([g, dx, x])
end

牛顿-拉夫法

matlab实现牛顿-拉夫法

matlab代码如下：

dx=1;                   % Change in variable is set to a high value 变量设置一个高值
x=input('Enter the initial estimate -> ');       % Initial estimate 初始估计值
iter = 0;                                       % Iteration counter    迭代计数
disp('iter   Dc        J         dx         x')% Heading for result 解的题头
while abs(dx) >= 0.001 & iter < 100          % Test for convergence 测试收敛
iter = iter + 1;                                % No. of iterations 迭代次数
Dc=4 - (x^3-6*x^2+9*x);                                  % Residual 差
J = 3*x^2-12*x+9;                                      % Derivative 导数
dx= Dc/J;                                       %Change in variable 修正量
x=x+dx;                                       % Successive solution 连续迭代
fprintf('%g', iter), disp([Dc, J, dx, x])
end
