正则表达式规则（续）

字符类:[ ]表示有一系列字符

  1. 只要包含有a或者 包含有b 或者 包含有c 就为true

    var reg =/[abc]/;

  2. 三选一：只有单独一个a，单独一个b或者单独一个c，才为真

    var reg = /^[abc]$/;

  3. 26个小写英文字符任何一个都返回true

    var reg = /^[a-z]$/

字符组合

  1. 26个英文字母大小写都可以，0-9任何一个数据都可以，-_也可以

    var reg =/^[a-zA-Z0-9]$/

  1. 如果中括号里边有^那就是取反，即不包含于

    var reg = /^[^a-zA-Z0-9]/

量词符号

量词 说明
* 重复零次或更多次
+ 重复一次或更多次
？ 重复零次或一次
{n} 重复n次
{n,} 重复n次或更多次
{n,m} 重复n到m次

具体使用

  1. a可以出现任意次

    var reg = /^a*$/;

  2. a可以出现一次或者很多次
    var reg1 = /^a+$/;

  3. a出现0次或者1次

    var reg1 = /^a?$/;

  4. a出现3次

    var reg1 = /^a{3}$/;

  5. a出现3次或者多次

    var reg1 = /^a{3,}$/;

  6. a出现3次并且小于10次

    var reg1 = /^a{3,10}$/;

量词设定某个模式出现的次数

  1. 这个模式用户只能输入英文字母，数字，下划线，短横线但是有边界符号和[]这就限定了只能多选1{6, 16}中间不要有空格

var reg = /^[a-zA-Z0-9_-]{6,16}$/
