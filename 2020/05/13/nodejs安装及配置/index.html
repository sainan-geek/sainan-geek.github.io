window10下nodejs安装及配置

1.下载

国内官网下载地址：http://nodejs.cn/download/

选择对应的版本下载就好

2.安装

点击下载好的安装包，如下页面出现后。先点击Next。

接受协议

选择安装位置，我这里选择默认的。

直接点Next

我这里没有勾选，后续需要什么模块，我自己安装。

安装完成后打开命令行窗口，输入node -v测试是否安装成功。

3.配置

npm默认下载地址在国外，为了下载模块时更快。要进行一个第三方模块nrm的下载。nrm可以进行npm下载地址的切换。

  1. 先下载使用npm install nrm -g下载。（-g是全局安装）

  2. 切换可用下载地址列表 nrm ls

  3. 切换npm下载地址 nrm use 下载地址名称，我这里用淘宝

    done.
