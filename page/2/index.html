JavaScript中||{}的意思

前言今天看到一段JavaScript代码不知道是什么意思,代码如下: var NETWORK = NETWORK || {};解释先看这个||符号，在js里面,||运算符，有个很有意思的用处： 比如（A||B）系统先判断A表达式的布尔值，是真是假。如果为A是真，直接返回A。如果A为假，直接返回B（不会判断B是什么类型）。 所以，var NETWORK = NETWORK || {};这样写的意思是，先判断NETWORK的值是否为真（也就是判断NETWORK本身是否存在），如果存在，则返回NETWORK。如果NETWORK不存在，则返回后面的{}。最后再将值返回给NETWORK。 这样写，虽然看着比较复杂点，但是效率高，特别是判断一些变量，属性或者是传参是否存在，如果存在就直接用，不存在，就赋予一个空变量。相当于下面的简写： if（NETWORK){ var NETWORK=NETWORK; } else{ var NETWORK={} //将一个空对象传给a }参考链接知乎问题参考

java内存的划分

java内存的划分 栈(Stack）：存放的都是方法中的局部变量。方法的运行一定要在栈中运行。 局部变量：方法的参数或方法{}内部的变量。 作用域：一旦超出作用域，立即从栈中消失。 堆(heap)：凡是new出来的东西都在堆中。 堆内存里面的东西都有一个地址值：16进制。 堆内存里面的数据，都有默认值。规则：   如果是整数类型   默认为0   如果是浮点类型   默认为0.0   如果是字符类型   默认为’\u0000’   如果是布尔类型   默认为false   如果是引用类型   默认为null 方法区(Methode Area)：储存.class相关信息，包含方法的信息。 本地方法栈(Native Area)：与操作系统相关。 寄存器(pc Register)：与cpu相关。
